
# 共情机器人

---

## 1. 触发器节点

| 节点原名                   | 新名称    | 节点功能解释                                      |
| ---------------------- | ------ | ------------------------------------------- |
| Schedule Trigger (30s) | 共情感知脉冲 | 系统的“生物钟”。每隔30秒自动激活一次全感官扫描，让机器人定期观察和倾听主人的状态。 |

---

## 2. 感知器节点

| 节点原名                    | 新名称    | 节点功能解释                        |
| ----------------------- | ------ | ----------------------------- |
| httpRequest - 摄像头输入（抓拍） | 视觉环境快照 | 调用摄像头硬件，捕捉当前主人的面部表情图像。        |
| httpRequest - 麦克风输入（录音） | 语音音频拾取 | 调动麦克风硬件，录制主人的语音片段，捕捉声音中的情绪信息。 |

---

## 3. 处理器节点

| 节点原名                         | 新名称       | 节点功能解释                                               |
| ---------------------------- | --------- | ---------------------------------------------------- |
| Set Camera Payload           | 图像规格标准化   | 将摄像头抓拍的原始图片进行转码和尺寸调整，使其符合AI视觉模型的分析标准。                |
| httpRequest - yolov8 手势/微笑识别 | AI 面部特征分析 | 核心视觉分析单元。利用算法识别主人是否在微笑或展现出特定的面部表情特征。                 |
| Set 识别-string（视觉）            | 视觉情绪状态标记  | 数据登记单元。将视觉分析得出的结果（如“检测到微笑”）存入系统变量。                   |
| httpRequest - ASR 语音识别       | 语音转文字引擎   | 将主人录制的音频流实时翻译为纯文字内容。                                 |
| Set 识别-string（ASR文本）         | 原始话语存证    | 将ASR翻译出的文字记录下来，供后续的情感分类模型分析。                         |
| httpRequest - BERT 情绪分类      | 文本情感倾向分析  | 深度NLP单元。利用BERT模型分析文字背后的情绪倾向，判断主人心情是否“难过”。            |
| Set 识别-string（文字）            | 文本情绪状态标记  | 数据登记单元。将文字分析得出的情绪结果（如“文字表现难过”）存入系统变量。                |
| Set 微笑响应 / 难过响应              | 共情行为编排    | 策略合成单元。根据不同情绪，打包组合一套对应的“表情包ID + 肢体动作码 + 语音Prompt模板”。 |
| httpRequest - LLM 文本回复       | AI 共情逻辑大脑 | 核心决策单元。基于“共情者”人设，根据主人的具体话语动态生成温暖的安慰或互动文字。            |
| Set LLM-string（用于TTS）        | 回复话术封装    | 准备单元。将LLM生成的回复文本整理后发送给语音合成模块。                        |
| httpRequest - TTS 文本         | 拟人语音合成    | 表达转换单元。将共情回复实时合成为带情感色彩的语音信号。                         |
| Set audioUrl                 | 音频资源定位    | 播放准备单元。获取生成语音文件的地址供喇叭播放。                             |

---

## 4. 逻辑 / 判断器节点

| 节点原名   | 新名称      | 节点功能解释                          |
| ------ | -------- | ------------------------------- |
| IF 是高兴 | 视觉正面情绪闸门 | 视觉决策开关。如果检测到主人在微笑，则激活“快乐互动”分支。  |
| IF 是难过 | 文本负面情绪闸门 | 语义决策开关。如果分析出主人情绪悲伤，则激活“深度安慰”分支。 |

---

## 5. 执行器节点

| 节点原名                    | 新名称    | 节点功能解释                              |
| ----------------------- | ------ | ----------------------------------- |
| httpRequest - 屏幕emoji显示 | 面部表情投射 | 物理显示单元。在屏幕上显示对应情绪的Emoji眼神（开心 / 难过）。 |
| httpRequest - 机械臂动作执行   | 肢体语言传动 | 物理动作单元。驱动机械臂完成安慰或欢呼等肢体反馈动作。         |
| httpRequest - 喇叭播放音频    | 语音合成播报 | 声音表达单元。通过喇叭播放共情语音，实现听觉互动。           |
