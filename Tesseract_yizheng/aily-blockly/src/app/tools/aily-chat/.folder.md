Presentation note:
- `aily-chat.component.scss` 现在承担 wireframe operator-console 壳层语法：技能卡、消息流、资源托盘、输入区和 token 仪表都必须复用 `spatial-design-ref.scss` token，不再单独引入渐变/glass 皮肤。
- `aily-chat.component.scss` 也承担 chat 局部圆角 token override；x-dialog viewer 中若仍写死 `999px` 或 `var(--radius-lg)` 大圆角，会直接破坏当前 AI chat 的矩形语法。
- `components/x-dialog/*viewer*` 与旧 `components/aily-*viewer*` 的按钮、标签、卡片、chip 都必须在组件内显式落到 `var(--radius-base)` 或 `var(--radius-sm)`；不要把“低圆角”只寄托在宿主变量覆写上。
