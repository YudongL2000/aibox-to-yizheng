
# 老付老刘 Case

---

## 1. 触发器节点

| 节点原名                   | 新名称       | 节点功能解释                                      |
| ---------------------- | --------- | ------------------------------------------- |
| Schedule Trigger (30s) | 定期开启新一轮任务 | 系统的“起搏器”。每隔30秒自动发出一个启动信号，驱动机器人开始新一轮的寻找目标任务。 |

---

## 2. 感知器节点

| 节点原名                    | 新名称    | 节点功能解释                                  |
| ----------------------- | ------ | --------------------------------------- |
| httpRequest - 摄像头输入（抓拍） | 视觉影像捕获 | 调用摄像头硬件，在信号触发的瞬间截取一张现场照片，作为系统后续分析的原始素材。 |

---

## 3. 处理器节点

| 节点原名                    | 新名称      | 节点功能解释                                   |
| ----------------------- | -------- | ---------------------------------------- |
| Set Random n (1..3)     | 随机行为计算   | 逻辑生成单元。通过内部算法产生一个随机数，为机器人随机生成下一步的探索行为意图。 |
| Set Image Payload       | 数据包预处理   | 翻译单元。将感应器抓拍的图片转码并打包成AI大脑可解析的协议格式。        |
| httpRequest - yolov8 识别 | AI 特征识别器 | 核心分析单元。利用视觉模型分析图片，从中寻找并标注老刘或老付的身份特征。     |
| Set Identify Result     | 识别状态登记   | 数据存储单元。将AI分析得出的姓名结果写入系统临时内存。             |
| Set 老刘 / 老付 动作与文案       | 互动脚本合成   | 内容编排单元。将识别到的身份转化为具体的动作指令与对应对话文案。         |
| httpRequest - TTS 生成音频  | 语音合成算法   | 表达转换单元。将文字脚本实时合成为可播放的音频流。                |
| Set Audio URL           | 播放地址封装   | 链路桥接单元。提取音频文件路径并传递给播放节点。                 |

---

## 4. 逻辑 / 判断器节点

| 节点原名            | 新名称    | 节点功能解释                                |
| --------------- | ------ | ------------------------------------- |
| IF n==1 / 2 / 3 | 寻人轨迹闸门 | 多路路径开关。根据随机行为结果选择具体的底盘运动路线。           |
| IF 是老刘 / 是老付    | 身份验证网关 | 条件分流核心。根据识别出的姓名决定进入惩罚模式（老刘）或欢迎模式（老付）。 |

---

## 5. 执行器节点

| 节点原名                                           | 新名称    | 节点功能解释                         |
| ---------------------------------------------- | ------ | ------------------------------ |
| httpRequest - Base Rotate / Forward / Backward | 底盘动力驱动 | 物理位移单元。控制轮式底盘执行旋转、前进或后退动作。     |
| httpRequest - 机械手执行手势                          | 手势肢体传动 | 物理交互单元。驱动机械手完成指定手势动作（如竖中指、比V）。 |
| httpRequest - 喇叭播放音频                           | 音频回响器  | 物理反馈单元。通过扬声器播放语音合成内容，完成语言互动。   |
