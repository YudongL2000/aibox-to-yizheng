# n8n 手动启动指南

## ⚠️ 重要：Node.js 版本要求

**n8n 要求 Node.js 版本：>=20.19 <= 24.x**

当前检测到的版本：v25.2.1（不兼容）

### 解决方案

#### 方案1: 安装并使用 nvm（推荐）

1. **安装 nvm**：
   ```bash
   curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
   ```

2. **重新加载终端**或执行：
   ```bash
   source ~/.bashrc
   # 或
   source ~/.zshrc
   ```

3. **安装并使用 Node.js 24**：
   ```bash
   nvm install 24
   nvm use 24
   ```

4. **验证版本**：
   ```bash
   node --version  # 应该显示 v24.x.x
   ```

5. **重新安装依赖并启动**：
   ```bash
   cd /Users/liuxiaolin/Desktop/n8n_base/n8n-master
   pnpm install
   pnpm dev
   ```

#### 方案2: 使用 Homebrew 安装 Node.js 24

```bash
brew install node@24
brew link --overwrite node@24
```

#### 方案3: 从官网下载安装

访问 https://nodejs.org/ 下载 Node.js 24.x LTS 版本

## 方法1: 使用 pnpm start（推荐）

在项目根目录执行：
```bash
cd /Users/liuxiaolin/Desktop/n8n_base/n8n-master
pnpm start
```

或者在 CLI 包目录执行：
```bash
cd packages/cli
pnpm start
```

## 方法2: 直接运行 bin/n8n

```bash
cd packages/cli
./bin/n8n start
```

## 方法3: 使用 Node.js 直接运行

```bash
cd packages/cli
node bin/n8n start
```

## 启动参数

### 自动打开浏览器
```bash
./bin/n8n start --open
# 或
./bin/n8n start -o
```

### 指定端口
```bash
N8N_PORT=3000 ./bin/n8n start
```

### 查看帮助
```bash
./bin/n8n start --help
```

## 检查启动状态

### 检查端口是否监听
```bash
lsof -i :5678
```

### 检查进程
```bash
ps aux | grep n8n
```

### 测试 HTTP 响应
```bash
curl http://localhost:5678
```

## 常见问题

0. **Node.js 版本不兼容**：
   - 错误信息：`Your Node.js version X.X.X is currently not supported`
   - 解决：使用 nvm 切换到 Node.js 20-24 版本
   - 检查版本：`node --version`

1. **如果端口被占用**：
   - 检查是否有其他 n8n 实例在运行：`lsof -i :5678`
   - 使用不同端口：`N8N_PORT=3000 ./bin/n8n start`

2. **如果编译未完成**：
   - 确保 dist 目录存在：`ls packages/cli/dist`
   - 重新编译：`cd packages/cli && pnpm build`

3. **数据库初始化**：
   - 首次启动会自动创建 SQLite 数据库
   - 数据库位置：`~/.n8n/database.sqlite`

4. **查看日志**：
   - 日志位置：`~/.n8n/logs/`
   - 或在启动时查看终端输出
