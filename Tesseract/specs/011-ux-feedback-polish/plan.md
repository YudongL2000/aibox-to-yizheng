## Technical Context

**Language / Version**: TypeScript 5 (backend + aily-blockly) · Dart 3 (frontend - NOT touched this sprint)  
**Primary Dependencies**: Express/WS (backend), Angular 19 + ng-zorro (aily-blockly), Electron 33 (desktop shell)  
**Storage**: Local JSON files under `backend/data/` · localStorage (dialogue mode state in aily-blockly) · in-memory session via `SessionService`  
**Testing**: Vitest (backend) · Angular/Karma (aily-blockly)  
**Target Platform**: Electron on Windows/macOS desktop  
**Project Type**: Multi-module monorepo desktop app  
**Performance Goals**: Diagram renders < 500ms after `summary_ready` event; mode switch < 200ms  
**Constraints**: No schema changes to existing session JSON; no new database tables; no breaking change to backend API contract  
**Scale/Scope**: ~12 files changed across 2 modules (backend + aily-blockly)
