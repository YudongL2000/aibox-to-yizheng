# 石头剪刀布

---

## 1. 触发器节点

| 节点原名(notes.title)                      | 新名称(notes.title)     | 节点功能解释(notes.subtitle)                               |
| ------------------------- | ------- | ------------------------------------ |
| Schedule Trigger - 5s定时运行 | 游戏开始触发器 | 游戏的“发令枪”。每隔5秒自动开启一轮新的游戏流程，确保游戏能循环进行。 |

---

## 2. 感知器节点

| 节点原名                | 新名称  | 节点功能解释                   |
| ------------------- | ---- | ------------------------ |
| httpRequest - 摄像头抓拍 | 用户快照 | 调用摄像头，在倒数结束的瞬间捕捉用户的手势画面。 |

---

## 3. 处理器节点

| 节点原名                       | 新名称      | 节点功能解释                                    |
| -------------------------- | -------- | ----------------------------------------- |
| Set Countdown              | 倒数计时设定   | 参数初始化单元。设定游戏开始前的倒数数字（如3, 2, 1）。           |
| httpRequest - TTS 倒数       | 倒数语音合成   | 表达转换单元。将“3, 2, 1”文字实时转化为语音文件，用于播报。        |
| Set Countdown Audio URL    | 倒数语音包封装  | 链路准备单元。获取倒数语音的播放地址，供扬声器调用。                |
| Set Robot n (1..3)         | 随机出拳计算   | 机器人“出拳”决策单元。利用随机算法在1-3之间选一个数，作为机器人本次出的招式。 |
| Set Robot=石头/剪刀/布          | 机器人意图登记  | 数据记录单元。将随机生成的数字翻译成具体的游戏手势标签，存入内存。         |
| Set Snapshot Payload       | 图像规格标准化  | 翻译单元。将抓拍的用户手势图片转码为AI模型可读的标准格式包。           |
| httpRequest - yolov8 手势识别  | AI 手势识别器 | 核心视觉分析单元。利用模型分析图片，判断用户出的是石头、剪刀还是布。        |
| Set Identify (userGesture) | 用户出拳结果登记 | 数据登记单元。将AI认出的用户手势标签存入系统，准备与机器人结果进行PK。     |
| Set Win/Lose/平局 -> 情绪      | 对战脚本合成   | 策略编排单元。根据输赢结果，匹配对应的表情（开心/难过）和对应的回复文字。     |
| httpRequest - TTS 结果文本     | 结果语音合成   | 表达转换单元。将“我赢啦”或“算你厉害”等文本实时合成音频流。           |
| Set Result Audio URL       | 结果语音包封装  | 链路桥接单元。提取结果音频地址，准备发给执行器播报。                |

---

## 4. 逻辑 / 判断器节点

| 节点原名                      | 新名称     | 节点功能解释                                       |
| ------------------------- | ------- | -------------------------------------------- |
| IF n==1 / 2 / 3           | 机器人出拳分拣 | 内部逻辑开关。根据随机计算的结果，把流程引向对应的机器人动作分支。            |
| IF 识别为空 / 平局 / Win / Lose | 胜负裁判网关  | 核心逻辑枢纽。对比机器人和用户的两个标签，根据游戏规则（如石头胜剪刀）判定最终胜负方向。 |

---

## 5. 执行器节点

| 节点原名                     | 新名称  | 节点功能解释                           |
| ------------------------ | ------ | -------------------------------- |
| httpRequest - 喇叭播放倒数     | 倒数音频播报 | 物理反馈单元。通过喇叭播放倒数语音，提醒用户准备出拳。      |
| httpRequest - 机械手出拳      | 物理手势驱动 | 物理动作单元。驱动机械手实时摆出对应的石头、剪刀或布的形状。   |
| httpRequest - 屏幕显示 emoji | 胜负表情展示 | 物理显示单元。根据裁判结果，在屏幕上显示出机器人此时的情绪眼神。 |
| httpRequest - 喇叭播放结果     | 结果音频播报 | 物理表达单元。最后播放出语音反馈，完成本轮游戏交互。       |
