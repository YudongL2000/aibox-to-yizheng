# 启动 n8n 服务器步骤

## 🔍 当前状态
- ❌ 后端服务器未运行（端口 5678 未被占用）
- ✅ Node.js 版本正确（v24.13.0）
- ✅ pnpm 版本正确（10.22.0）

## 🚀 启动步骤

### 步骤 1：打开终端，进入项目目录

```bash
cd /Users/liuxiaolin/Desktop/n8n_base/n8n-master
```

### 步骤 2：确认环境

```bash
# 确认 Node 版本
node -v
# 应该显示：v24.13.0

# 如果不是，加载 nvm
source ~/.bashrc
nvm use 24
```

### 步骤 3：启动开发服务器

```bash
pnpm dev
```

### 步骤 4：等待启动完成

**首次启动需要 3-5 分钟**，等待看到以下输出：

```
✅ n8n ready on 0.0.0.0, port 5678
✅ VITE v5.x.x  ready in xxx ms
✅ ➜  Local:   http://localhost:8080/
```

### 步骤 5：验证服务器运行

**打开新的终端窗口**（不要关闭运行 `pnpm dev` 的终端），执行：

```bash
# 检查后端服务器
curl http://localhost:5678

# 检查前端服务器
curl http://localhost:8080
```

如果返回 HTML 内容，说明启动成功！

## ⚠️ 常见问题

### 问题 1：启动很慢
- **原因**：首次启动需要编译所有包
- **解决**：耐心等待 3-5 分钟

### 问题 2：编译错误
- **检查**：终端中的错误信息
- **解决**：可能需要先运行 `pnpm install`

### 问题 3：端口被占用
```bash
# 查找占用端口的进程
lsof -i :5678
lsof -i :8080

# 停止进程（替换 <PID>）
kill -9 <PID>
```

### 问题 4：依赖问题
```bash
# 重新安装依赖
pnpm install

# 重新构建
pnpm build
```

## 📝 快速命令参考

# 仅启动后端
pnpm dev:be
