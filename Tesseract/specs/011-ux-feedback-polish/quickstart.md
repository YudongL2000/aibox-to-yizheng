# Quickstart: 011 UX Feedback Polish

## Backend

1. 进入 `backend/` 并启动 agent 服务。
2. 在教学模式提交完整需求，例如“我想控制一个舵机在检测到人脸时旋转 90 度”。
3. 期望：直接进入 `summary_ready`，不再重复澄清。

## Chat UX

1. 在教学模式完成一轮工作流。
2. 切到对话模式。
3. 期望：历史消息与教学模式隔离；顶部立即看到技能卡片；AI 名称显示为 `Tess`。

## Mermaid

1. 在教学模式提交完整工作流描述。
2. 期望：`summary_ready` 阶段先出现初步流程图，`workflow_ready` 阶段出现完整流程图。
3. 期望：不出现 `No diagram type detected`。

## Hotplug

1. 进入热插拔步骤。
2. 期望：不再出现“模拟插入”文案。
3. 接入真实硬件或硬件桥快照到达后，期望：状态清单自动变为已接入并自动推进当前节点。
4. 若 30 秒内未检测到，期望：出现超时提示，并提示可检查连接或跳过当前步骤。

## Desktop Polish

1. 检查窗口标题：应显示 `Tesseract`。
2. 检查顶部左侧：应显示硬件连接状态框。
3. 打开“我的库”：尺寸应比旧版本更小。
4. 打开数字孪生：首次加载前应看到骨架屏，而不是灰色空白。
